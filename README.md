# Runny
## Description
- This project implements a sensor suite for capturing the characteristics regarding the soil health of potted plants (with a scaled goal of tracking the same characteristics in lawn/field run-off); the selected sensor collection includes monitoring of the following properties:
	- Nitrogen, phosphorus, and potassium
	- Local humidity and temperature
	- Moisture level
